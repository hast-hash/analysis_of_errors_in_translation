📋 Table 2 相当：評価者間一致度（AD指数）

Interrater agreement — Average Deviation (AD) index (cf. Table 2 in Long 2022)
Long (2022) Table 2はMirai翻訳の精度評価（10点スケール）の評価者平均値とAD指数を示す。本分析では、SCATEタクソノミーのフルエンシー評価（エラー数）に基づき、テキスト × モデルごとに評価者のエラー総数を集計し、AD指数（= Σ|xi − x̄| / n）で評価者間の一致度を算出した。AD指数が小さいほど評価者間の一致度が高いことを示す。
